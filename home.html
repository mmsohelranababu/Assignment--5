Important FAQ

What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface for web documents, representing the structure of a webpage so it can be manipulated with scripts like JavaScript.

How do you select an element from the DOM?

You can select a DOM element using document.getElementById('id') or document.querySelector('selector'). For multiple elements, use document.querySelectorAll('selector').

What is event delegation in the context of the DOM, and why is it useful

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events on its child elements. It is useful because it improves performance by reducing the number of event listeners and allows handling dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute(), getAttribute(), or by directly modifying properties like element.id. To change styles, modify the style property, e.g., element.style.color = 'red'.
